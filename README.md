Golem
=====

Prerequisites
-------------
* Java
* ElasticSearch


Install
-------
    sudo npm install --global mocha
    npm install
    npm test

Dev
---
    mocha [$fie]
    eslint
    git commit -am $what

Roadmap
-------
1. Initial client, to fetch public page stream
1. Initial index
1. Initial term search
1. Client to reverse pps to individual
1. Search culpable individuals
1. Refactor for second client
